# Typescript, Koa, Vue Template

Vue + Koa in TypeScript by Babel 7.

## Requirement

* Node.js >= 12

## Usage

### Installation

``` sh
git clone git@github.com:m80126colin/typescript-koa-vue-template.git
cd typescript-koa-vue-template/
yarn
```

### Development

``` sh
yarn dev
```

### Build and Run

``` sh
yarn start
```
